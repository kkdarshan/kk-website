Git Cheat Sheet for Beginners

Initialize Repository

git init

Creates a new Git repository in the current directory.

Check Status

git status

Shows changes staged, unstaged, and untracked files.

Add Files

git add <filename>

Add specific files to staging area before commit.

Commit Changes

git commit -m "Your message here"

Record changes to the repository with a descriptive message.

Push to Remote

git push origin main

Sends committed changes to the remote repository (GitHub).

Clone Repository

git clone <repository_url>

Copies an existing repository from remote to local machine.

View Log

git log --oneline

Shows commit history in a compact form.
